# Tugas Praktikum Pemrograman Berorientasi Objek (Java)

Repisitori ini berisi kumpulan kode program Java untuk memenuhi tugas praktikum mengenai Class, Object, Inheritance, Polymorphism, dan Abstraksi.

## Deskripsi Kode

### 1. Matematika & MobilLengkap
* **Matematika**: Berisi method operasi aritmatika (tambah, kurang, kali, bagi) dengan dua parameter.
* **MobilLengkap**: Simulasi aksi dasar kendaraan seperti menghidupkan mesin, mengubah gigi, dan mematikan mesin.

### 2. Inheritance (Pewarisan)
* **Hewan**: Implementasi hubungan hierarki antara class induk (Hewan) ke subclass (Herbivora & Karnivora) hingga ke detail jenis hewan (Kelinci).

### 3. Polymorphism
* **Overriding**: Implementasi method `gambar()` pada class `Bentuk` yang diubah perilakunya oleh class `Segitiga` dan `Persegi`.
* **Overloading**: Penggunaan nama method `tambah()` yang sama dengan jumlah atau tipe parameter yang berbeda pada class `Hitung`.

### 4. Abstraksi
* **Abstract Class**: Implementasi class abstrak `MakhlukHidup` dan `Kendaraan` yang mewajibkan subclass untuk mendefinisikan method `bernapas()` dan `bergerak()`.
